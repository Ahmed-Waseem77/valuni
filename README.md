# valuni
Software Engineering project for Fall 2023-2024 AUC, Online University Evaluation System 

## How to setup
 1. run the setup.py script in root directory 
    ``` 
    python3 setup.py 
    ```
 2. set up expo development environment in working directory ``valuni_wd``
    ```  
    cd valuni_wd
    npx expo start --web 
    ```
